# API Health Monitoring - Datadog

## Monitoring Tool

Datadog Synthetic Monitoring

## API Information

### API Name
Shopist API

### Base URL
https://api.shopist.io

### Description

Shopist API is an e-commerce REST API used for managing products, carts, and related shopping workflows. For this assignment, Datadog Synthetic Monitoring has been configured to continuously monitor API availability, response time, and reliability. :contentReference[oaicite:0]{index=0}

---

# Objective

The objective of this monitoring setup is to proactively detect:

- API downtime
- Slow response times
- Availability issues
- Unexpected HTTP response codes
- Performance degradation
